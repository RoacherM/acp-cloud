# Agent Skills

This workspace provides three skills for media generation and analysis.

## Available Skills

### Seedance Image Generator
Generate images using Seedance 5.0 Lite. See `.codex/skills/seedance-image/SKILL.md` for usage.

### Mureka Music Generator
Generate music (lyrics + audio) using Mureka AI. See `.codex/skills/mureka-music-generator/SKILL.md` for usage.

### Gemini Video Analyzer
Analyze video content using Gemini. See `.codex/skills/gemini-video/SKILL.md` for usage.

## Important
- Read the SKILL.md file for each skill before using it
- Check that the corresponding `.env` file exists in the skill's `scripts/` directory
- Follow the exact commands specified in each SKILL.md
